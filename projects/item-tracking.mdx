## Project overview

Our cloud solutions enabled the efficient tracking of asset delivery's around the UK.
Sensors deployed sending heartbeats to gateways which show's the location of the asset. Dealing with edge telemetry and performing queries to show the status and location of all the assets.

## Technology used

- Azure Devops
- .Net5
- C#
- Postgres
- Typescript
- GraphQL
- Docker
- Kubernetes
- Azure (IoT Hub, Iot Edge, Cosmos DB, Blog Storage, Event Grid/ Hub)
