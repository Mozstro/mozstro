## Project overview

Helping a supermarket track any potential loss of inventory on shelves using machine vision. Dealing with edge telemetry that is stored as time series data then agregated and displayed in a user interface which allows the client to see in real time as theft is taking place. The system can also alert guards of suspicious actions.

## Technology used

- Azure Devops
- .Net5
- C#
- Postgres
- Typescript
- GraphQL
- Docker
- Kubernetes
- Azure (IoT Hub, Iot Edge, Cosmos DB, Blog Storage, Event Grid/ Hub)
